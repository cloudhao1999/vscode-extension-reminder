# 下班了吗 README

一个社畜人在周末加班的时候为了学习新技术写的一个小插件，用来判断是否下班了。

![setting](images/setting.gif)

## Features

- [x] 可自定义下班时间
- [x] 底部状态栏显示剩余时间

## Release Notes

### 1.0.0

实现基本功能，可自定义下班时间，底部状态栏显示剩余时间。

### 1.0.1

修复了一个小 bug。

### 1.1.0

可自定义上下班时间。
